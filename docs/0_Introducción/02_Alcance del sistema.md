# Alcance del sistema 

- **Centralizar** la gestión de cuentas institucionales y servicios que la DTI ofrece. 

- **Automatizar** procesos de creación, transferencia, inhabilitación, habilitación, eliminación y actualización de las cuentas de Directorio Activo y asignación de servicios. 

- **Integrarse** con servicios de directorio activo, cuentas de correo y administración de base de datos. 

- **Mejorar** la seguridad, control y capacidad de identificar y seguir el rastro de una cuenta con todos sus movimientos y servicios.
